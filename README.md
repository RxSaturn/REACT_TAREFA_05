## Contexto Escolhido

**Contexto A: Dashboard de Gestão de Atividades**

Interface voltada para a organização de tarefas pessoais ou profissionais.

### Funcionalidades Implementadas:
- **Lista de Tarefas:** Exibe tarefas com título, categoria, nível de prioridade (Alta, Média, Baixa) e status (concluído ou pendente)
- **Renderização Condicional:**
  - Tarefas com prioridade "Alta" possuem estilização visual de alerta (bordas vermelhas) via CSS Modules
  - Etiqueta "Urgente" exibida apenas para tarefas de alta prioridade ainda não concluídas
- **Comunicação via Props:** Arrow Function passada via props que exibe os detalhes da tarefa ao clicar no botão "Ver Detalhes"
